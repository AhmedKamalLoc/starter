# ActionDialog

The ActionDialog component is a customizable dialog used for confirmation actions, typically
destructive operations like deletion.

## Usage

```TypeScript
import { ActionDialog } from '@/components/action-dialog/action-dialog';
```

## Props

| Prop             | Type                    | Description                                   |
| ---------------- | ----------------------- | --------------------------------------------- |
| title            | string                  | The title of the dialog                       |
| description      | string                  | The description text                          |
| cancelButtonText | string                  | Text for the cancel button                    |
| action           | React.ReactNode         | The action button/content                     |
| icon             | React.ReactNode         | Icon displayed in the dialog (default: Trash) |
| children         | React.ReactNode         | content to display                            |
| isOpen           | boolean                 | Controls the open state of the dialog         |
| onOpenChange     | (open: boolean) => void | Callback when open state changes              |
| onCancel         | () => void              | Callback when dialog is cancelled             |
| trigger          | React.ReactNode         | Custom trigger button component               |
| footerClassName  | string                  | Custom class for the footer                   |
| separator        | React.ReactNode         | separator for the footer                      |
| ref              | RefObject               | ref for the content container                 |
